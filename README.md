# Magnetic-Field-Visualization-Project
🧲 **Magnetic Field Visualization | HTML5 Canvas + JavaScript**

Excited to share my latest frontend project — **Magnetic Field Visualization**, an interactive physics-inspired animation built using **HTML5 Canvas, CSS3, and Vanilla JavaScript**.

### ✨ Key Features

🔹 Interactive mouse-controlled magnetic attraction
🔹 450+ animated particles with smooth motion
🔹 Dynamic glowing field-line connections
🔹 Physics-based force calculations and vector mathematics
🔹 Motion trail effect for a modern visual experience
🔹 Fully responsive and optimized performance
🔹 No external libraries used

### 🛠️ Technologies Used

• HTML5
• CSS3
• JavaScript (ES6)
• HTML5 Canvas API
• Particle System
• Vector Mathematics
• requestAnimationFrame()

This project helped me explore real-time graphics rendering, animation loops, and interactive particle physics while creating a visually engaging user experience.
